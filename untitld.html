XHTML

XHTML is an acronym for extensible Hyper Text Markup Language.

The "x" came from XML or extensibie Mark-up Language. Though different from HTML, XHTML also uses the filename extension html.

The content of the actual coding might change because of XHTML conventions but it still holds the same filename extension.



Difference of HTML and XHTML

XHTML is stricter and cleaner version of HTML Aimed to replace HTML
All new Web browsers support XHTMI, because there came to a point that a huge amount of web pages have been written in the wrong syntax Since web browsers still force to interpret HTML the way they could, sometimes the wrong syntax could still display a logically correct output

  • Proper Nesting
    Proper nesting of elements has been exemplified and XHTMI elements must be properly nested Proper nesting of elements would mean that, for example if you were to make a piece of text appear in boldface and italics using < b>< /b> and < i>< /i> respectively You would need to close last the first tag you opened

    Recall the container tags Container tags have start and end tags. and end tags are defined by a slash End tags are used to basically close the tag XHTMI requires proper closing of tags

    All XHTML elements MUST be written in lowercase characters same as with the attributes.
